# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by emailing security@wolia.dev (or opening a private security advisory on GitHub).

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and work with you to understand and address the issue.

## Security Measures

- Dependencies are regularly audited with `cargo audit`
- Supply chain checks via GitHub Actions
- All PRs require passing CI checks
