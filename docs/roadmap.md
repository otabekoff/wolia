# Wolia Roadmap

## Phase 1: Foundation (Current)

### Engine Core
- [x] Document model (wolia-core)
- [x] Geometry types (wolia-math)
- [x] Layout engine (wolia-layout)
- [x] GPU renderer (wolia-render)
- [x] Edit system (wolia-edit)
- [x] Platform abstraction (wolia-platform)
- [x] Asset loading (wolia-assets)
- [x] Plugin system (wolia-plugin)

### File Formats
- [x] Native .wolia format
- [ ] DOCX import/export
- [ ] XLSX import/export
- [ ] PPTX import/export
- [ ] PDF export
- [ ] Markdown import/export

### Apps
- [ ] Wolia Write basic UI
- [ ] Wolia Grid basic UI
- [ ] Wolia Deck basic UI
